# ImFine 상세 구현 로드맵 (Detailed Implementation Roadmap)

이 로드맵은 개발자가 즉시 개발에 착수할 수 있도록 기술적인 준비 사항부터 배포까지 아주 세부적으로 구성되었습니다.

---

## Phase 0: 개발 환경 구축 및 기초 설계 (1~2주)
- [ ] **기술 스택 초기화**
    - [ ] React Native 프로젝트 생성 및 기초 폴더 구조 설계.
    - [ ] **브랜드 디자인:** 필기체 'IF' 로고 및 앱 아이콘 제작 (SVG/PNG).
    - [ ] **모션 구현:** Lottie 또는 React Native Reanimated를 이용한 스플래시 애니메이팅.
    - [ ] **Supabase Client 설정** (Auth, DB 연동).
- [ ] **데이터베이스 모델링 구현**
    - [ ] **Supabase Table Editor**를 이용한 User, Mood, Message, Point 테이블 생성.
    - [ ] 초기 시드 데이터(감정 이모티콘 리스트, 전문가 카테고리) 준비.
- [ ] **인증 시스템 연동**
    - [ ] Firebase Auth 또는 전용 OAuth 서비스를 이용한 Google, Apple, Kakao 로그인 구현.
    - [ ] JWT 기반 토큰 인증 로직 및 API 보안 적용.

---

## Phase 1: 핵심 기능 - 기분 기록 & 매칭 피드 (3~4주)
- [ ] **기분 기록 모달 (Entry Flow)**
    - [ ] 메인 진입 시 당일 기록 체크 로직 구현.
    - [ ] 이모티콘 선택 UI 및 한줄 일기 입력 폼 제작.
    - [ ] 익명 공유 토글 및 Safety Check 연장 API 연결.
- [ ] **개인화 메인 피드 (SNS Core)**
    - [ ] 무한 스크롤(Infinite Scroll) 기반의 카드형 피드 구현.
    - [ ] 사용자 기분 기반 매칭 쿼리(DB 수준) 및 추천 필터링 적용.
    - [ ] 댓글(Reply) 기능 및 하트(Like) 소통 기능 연동.
- [ ] **사용자 경험 고도화**
    - [ ] X/Threads 스타일의 부드러운 전환 애니메이션 효과 적용.
    - [ ] 텍스트 상세 보기 화면 및 작성자 익명 프로필 팝업.

---

## Phase 2: 소통 활성화 및 마음 보관함 (5~6주)
- [ ] **채택 및 포인트 시스템**
    - [ ] 댓글 '도움됨/채택' 기능 및 포인트 이동 트랜잭션 구현.
    - [ ] 포인트 획득 시 시각적인 애니메이션 효과(코인 이동 등) 적용.
    - [ ] 포인트 사용 내역 및 현재 잔액 조회 화면 개발.
- [ ] **전문가 인증 및 활동**
    - [ ] 전문가 자격증 업로드 및 승인 대기 시스템(Admin 기초).
    - [ ] 인증된 사용자의 블루 체크마크 및 특수 카드 UI 적용.
- [ ] **마음 보관함 (Heart Box)**
    - [ ] 텍스트 기반 사후 메시지 작성 및 편집 기능.
    - [ ] **보안:** 마음함 진입 시 로컬 생체 인증(FaceID/TouchID) 연동.
    - [ ] 발송 트리거(Inactivity Tracking) 서버 사이드 스케줄러(Node-cron 등) 구축.

---

## Phase 3: 미디어 서비스 및 수익화 (7~8주)
- [ ] **영상 예약 메시지**
    - [ ] 디바이스 카메라 연동 및 영상 녹화/압축 기능 구현.
    - [ ] AWS S3 - CloudFront 연동을 통한 안전한 영상 업로드 및 스트리밍.
- [ ] **기억의 초상 (AI Photo Editor)**
    - [ ] AI 이미지 필터링 및 리터칭 API 연동 (Cloudinary 또는 전용 AI 모델).
    - [ ] 저화질 사진 복구 및 화질 개선(Upscaling) 서버 로직 구축.
- [ ] **프리미엄 구독 및 결제**
    - [ ] Stripe 또는 RevenueCat을 이용한 인앱 결제(IAP) 시스템 구축.
    - [ ] 구독 플랜(A/B)에 따른 기능 제한(회수, 용량 등) 로직 적용.
- [ ] **포인트 환전 시스템**
    - [ ] 본인 확인 서비스(NICE 등) 연동을 위한 정밀 인증 모듈.
    - [ ] 계좌 정보 등록 및 환전 신청 관리 기능.

---

## Phase 4: 안정화 및 정식 출시 (9주~)
- [ ] **QA 및 보안 점검**
    - [ ] 메시지 오발송 방지를 위한 엣지 케이스 테스트.
    - [ ] 개인정보 암호화 저장 및 전송 구간 보안 점검.
- [ ] **스토어 등록 및 마케팅 준비**
    - [ ] App Store, Play Store 스토어 에셋(스크린샷, 설명서) 제작.
    - [ ] 초기 유입을 위한 마케팅 도구(Firebase Analytics 등) 연동.
